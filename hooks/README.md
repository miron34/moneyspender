# hooks/

Кастомные React-хуки. На текущем этапе пуста.

## Что сюда пойдёт

- Хуки, инкапсулирующие повторяющуюся UI-логику (например, `useSheetState`, `useDebouncedValue`).
- Хуки-обёртки над платформенным API (например, `useHaptics` с no-op на web).

## Что сюда не пойдёт

- Доступ к стейту (только через Zustand селекторы прямо в компонентах).
- Бизнес-логика (живёт в `store/` или `utils/`).
- Платформенная конфигурация (`constants/`).
