# Дизайн-система

Источник истины — `handoff/MoneySpender v4.html` и `handoff/README.md` (спецификация дизайнера).

## Цвета

| Токен | Значение | Применение |
|---|---|---|
| `bg` | `#07091a` | Фон экрана (самый глубокий) |
| `surface` | `#0d1124` | Карточки, ряды трат |
| `surfaceHigh` | `#131929` | Контейнеры пилюль, шиты |
| `surfaceTop` | `#1a2235` | Поля ввода, иконки топ-листа |
| `border` | `rgba(255,255,255,0.07)` | Тонкие границы карточек |
| `borderMid` | `rgba(255,255,255,0.11)` | Границы шитов сверху |
| `accent` | `#3b5bdb` | Синий акцент: FAB, активные пилюли, CTA |
| `text` | `#f0f2ff` | Основной текст |
| `textDim` | `rgba(240,242,255,0.65)` | Вторичный текст |
| `textMuted` | `rgba(240,242,255,0.35)` | Подписи, плейсхолдеры |
| `negative` | `#ff5757` | Расходы (всегда), удаление |
| `positive` | `#4caf50` | Положительные дельты в аналитике |

**Правила:**
- Один акцент — синий. Цвета категорий используются только точечно (иконки, бары, обводки).
- Расходы всегда отображаются как отрицательные `−X ₽` цветом `negative`.
- Полупрозрачные оверлеи поверх цветов: alpha `0x1a` (~10%) — фон иконки, alpha `0x22` (~13%) — пилюля категории, alpha `0x55` (~33%) — обводка пилюли.

## Типографика

**Шрифт:** Inter (300, 400, 500, 600, 700) через `@expo-google-fonts/inter`.

| Назначение | Размер | Вес | Letter-spacing |
|---|---|---|---|
| Крупные цифры (тотал) | 40–42 | 300 | -2 |
| Заголовки экранов | 20–22 | 600 | -0.5 |
| Заголовки карточек | 13 | 600 | 0 |
| Имена в строках | 13–14 | 500 | 0 |
| Тело / итоги | 12–14 | 400–500 | 0 |
| Подписи каплс | 11 | 300 | 0.04em (uppercase) |
| Мелкие подписи | 11–12 | 300–400 | 0 |

Цифры — намеренно тонкие (300) с отрицательным letter-spacing для эффекта премиальности.

## Компоненты-паттерны

### Pill (пилюля)

Универсальный компонент. Активная: `bg = accent`, текст белый. Неактивная: `bg = surfaceHigh`, текст `textMuted`. Фильтрационная активная: `bg = accent + 0x22`, граница `accent + 0x55`, текст `accent`.

### Card

`bg = surface`, `border 1px solid border`, `radius 16–18`, `padding 14–16`. Внутри — собственная иерархия: маленький подзаголовок капсом сверху → значение/контент.

### ExpenseRow

`bg = surface`, `border`, `radius 14` (compact) / `16` (full). Слева — цветной квадрат-иконка (`category.color + 0x1a` фон). Посередине — название + дата подписью. Справа — сумма красным с минусом.

### Period switcher

Контейнер: `bg = surfaceHigh`, `radius 10`, `padding 3`, `gap 2`. Каждая пилюля: `padding 5px 9px`, `radius 7`. Активная — синяя.
